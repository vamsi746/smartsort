About SmartSort

SmartSort is an AI-powered web application that helps identify the freshness of fruits and vegetables using deep learning and transfer learning (VGG16).

It assists farmers, vendors, and households in reducing food waste and improving quality control.

This project uses a custom-trained model to classify over 28 fruit and vegetable classes as either fresh or rotten. Users simply upload an image, and SmartSort instantly analyzes and displays the prediction.
